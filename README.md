# Smart Faculty Leave Management System

## Overview

A web application that automatically assigns substitute faculty when a teacher applies for leave by checking timetable availability.

## Features

- Faculty Login
- Leave Application
- Automatic Load Adjustment
- Timetable Management
- Admin Dashboard

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- Bootstrap

## Installation

```bash
git clone https://github.com/username/smart-load-adjustment.git

cd smart-load-adjustment

pip install -r requirements.txt

python app.py
